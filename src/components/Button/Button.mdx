# Button

## Usage

### Basic

<Playground>
  <Button onClick={() => alert('Variant is a primary!')}>
    <span>Click Me!</span>
  </Button>
  <Button variant="secondary" onClick={() => alert('Variant is a secondary!')}>
    <span>Click Me!</span>
  </Button>
</Playground>

### Advanced

<Playground>
